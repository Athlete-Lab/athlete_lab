# ATHLETE LAB - Sistema de Registro de Treinos

## Descrição

O **Athlete Lab** é um sistema de gestão esportiva com foco educacional, desenvolvido para armazenar, organizar e acompanhar informações de atletas de diferentes modalidades esportivas.

A plataforma permite o cadastro e gerenciamento de atletas, treinadores, equipes, esportes e treinos, além de possibilitar o acompanhamento do desempenho e evolução dos atletas ao longo do tempo.

O sistema foi desenvolvido como parte da disciplina **Linguagem de Programação Orientada a Objetos**, com o objetivo de aplicar conceitos de modelagem de sistemas, orientação a objetos e desenvolvimento de software.

---

## Objetivo do Sistema

O objetivo do sistema é integrar **gestão, treinamento e educação esportiva**, auxiliando treinadores na organização dos atletas e permitindo que os próprios atletas acompanhem sua evolução de maneira estruturada e eficiente.

O sistema também busca facilitar:

* Organização de treinos
* Acompanhamento de desempenho
* Gerenciamento de equipes
* Registro da evolução dos atletas
* Disponibilização de conteúdos educacionais

---

## Funcionalidades

### Para Atletas

* Cadastro e gerenciamento de perfil
* Visualização de informações pessoais
* Acesso ao histórico de treinos
* Acompanhamento de desempenho e evolução
* Acesso a conteúdos educacionais
* Realização de treinos personalizados

### Para Treinadores

* Visualização dos atletas sob sua responsabilidade
* Vinculação e desvinculação de atletas
* Organização de equipes
* Cadastro e atribuição de treinos
* Registro de evolução dos atletas
* Sugestão de melhorias no desempenho

### Para Administrador

* Gerenciamento geral do sistema
* Controle de dados dos usuários
* Organização de conteúdos educacionais
* Cadastro e manutenção de treinos prontos
* Garantia do funcionamento do sistema

---

## Entidades do Sistema

O sistema é composto pelas seguintes entidades principais:

* Atleta

* Esporte
